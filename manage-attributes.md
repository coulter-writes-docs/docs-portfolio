# Manage attributes
 
This guide explains how to create, edit, and delete custom attributes in WorkHop. Use attributes to classify groups of users across the platform. Only admins can create and manage custom attributes.

 
## Before you start
 
Before you manage attributes, make sure:
 
* You are a WorkHop administrator.
* If you want to edit or delete an attribute, make sure it is not used in any active reports.

 
## Create a custom attribute
 
1. Go to **Settings** and select **Attributes** from the side menu.
2. Select **Create an attribute**.
3. Enter a unique attribute name.
4. Select a data type: **Choice**, **Date**, **Number**, or **Text**.
5. If you selected **Choice**:
    1. Choose **New value** to add a selectable value.
    2. Enter at least 2 values for each **Choice** attribute.
        - For example, for an attribute named *Tenured*, you might add the values *Yes* and *No*.
6. Click **Save** to create the attribute.
 
Once created, custom attributes show on the **Attributes** page.
 

## Edit a custom attribute
 
1. Go to **Settings** and select **Attributes** from the side menu.
2. Locate and select the attribute you want to edit. Use the search bar to find it by name if needed.
3. Edit the attribute's name, values, and permissions as desired.
4. Click **Save** to apply your changes.
 

## Delete a custom attribute
 
>[!NOTE]
>You cannot delete an attribute used in an active report.
 
1. Go to **Settings** and select **Attributes** from the side menu.
2. Check the box next to each attribute you want to delete.
3. Click **Delete**, then confirm in the pop-up window.
