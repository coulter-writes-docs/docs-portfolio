# Release notes — WorkHop R2 2026

Release date: 2026-09-01

This release introduces AI-powered course suggestions for managers, along with improvements to attribute management and survey scheduling.

## New features

* **AI course suggestions for managers**

Managers can now generate AI-powered learning course recommendations for their team members from their **Dashboard**. Suggestions are based on each employee's role, attributes, and survey responses.

## Improvements

* **Bulk attribute deletion**

  HR administrators can now select and delete multiple custom attributes at once from the **Attributes** page. Previously, attributes could only be deleted one at a time.

* **Survey recurrence options**

  HR administrators can now set surveys to recur on a weekly, monthly, or quarterly basis when scheduling them. Previously, surveys could only be sent once and had to be manually rescheduled each cycle. Existing surveys are not affected by this change.

## Bug fixes

* **Fixed issue where survey notifications were not delivered to some users**

  Resolved an issue where some employees did not receive survey notification emails after accepting their invitation. Affected users will now receive notifications for all open surveys assigned to them.

* **Fixed issue where custom attribute values were not saved after editing**

  Resolved an issue where changes to created custom attributes were not saved after clicking **Save** on the **Attributes** page. Edited values now save correctly.

* **Fixed issue where manager meeting scheduler showed incorrect time zones**

  Resolved an issue where the meeting scheduler displayed available time slots in the organisation's default time zone rather than the logged-in user's local time zone. Meeting times now reflect each user's local time zone correctly.
